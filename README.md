# Computer Science Assignment and Project Template
This tex file is created as a template for teachers to use for projects and assignments. It provides documentation on how to edit certain specs, add questions, and add an answer key. Important commands like naming sections, adding solutions, creating tables, inserting images, adding links, and writing code are included.  

**Author:**
- Jenna Ramsey-Rutledge

## Quick Start

### Prerequisites

- Overleaf Account

### Compiling for the first time
1. Download the main.tex file
2. Compress file to ZIP File
3. Select New Project -> Upload Project
4. Click Recompile
5. Feel free to rename the template to the assignment or project
